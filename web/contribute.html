Item Image Contribution Page

This page shows all tile/item images used by the tool.
If an image is missing, it will show a placeholder and indicate the required filename.

To contribute:

  • Find an entry marked “Missing Image”
  • Create a 64x64 PNG file
  • Place it at the exact path shown (e.g. images/tiles/dirt_ground.png)
  • Submit a PR on GitHub
